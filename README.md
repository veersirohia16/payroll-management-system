# Payroll Management System

## Overview
A Python-based Payroll Management System that automates salary processing and payslip generation.

## Features
- Employee salary calculation
- Payslip generation
- Database connectivity
- Automated payroll processing
- Testing modules

## Technologies Used
- Python
- SQL Database
- Git & GitHub
